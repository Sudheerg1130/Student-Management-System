# 🎓 Student Management System (Java)

A simple and efficient console-based Java application to manage student records. This project demonstrates core Java concepts and basic data handling using the Collections Framework.

---

## 🚀 Features

- ➕ Add multiple students
- 📋 Display all student records
- 🔍 Search student by ID
- ✏️ Update student details
- ❌ Delete student record
- 🧭 Menu-driven user interface

---

## 🛠️ Technologies Used

- Java
- ArrayList (Java Collections Framework)
- Scanner (for user input)

---

## 📚 Concepts Covered

- Object-Oriented Programming (OOP)
- Constructors
- ArrayList operations
- Loops and conditional statements
- CRUD operations (Create, Read, Update, Delete)
